# E-Commerce-Sales
Madhav E-Commerce Sales Dashboard is an interactive Power BI dashboard that provides deep insights into sales performance, profit trends, customer behavior, and product category analysis across different states and time periods.

It helps businesses track:

📈 Monthly Profit Trends

🛒 Category & Sub-Category Performance

💳 Payment Mode Analysis

🌍 State-wise Sales Distribution

👥 Customer Purchase Insights

3️⃣ Tech Stack (Professional + Clean with Emojis)
🛠 Tech Stack

📊 Power BI – Data visualization & dashboard development

🗄 Excel / CSV – Data source & preprocessing

📐 DAX – Calculations & measures

🔄 Power Query – Data transformation

🧮 Data Modeling – Relationship management
